## ⚙️ Что делает утилита

Забудьте про ручной перебор и мёртвые прокси.

Программа:

* парсит список серверов из репозитория
* автоматически проверяет их работоспособность
* отбирает только **живые и актуальные**
* выводит готовый список для использования

Вам остаётся только:

1. Дождаться окончания проверки
2. Выбрать сервер из списка
3. Вставить его в настройки клиента
