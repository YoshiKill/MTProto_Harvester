## ⚙️ Что делает утилита

Забудьте про ручной перебор и мёртвые прокси.

Программа:

* парсит список серверов из репозитория
* автоматически проверяет их работоспособность
* отбирает только **живые и актуальные**
* выводит готовый список для использования

Вам остаётся только:

1. Дождаться окончания проверки
2. Выбрать сервер из списка
3. Вставить его в настройки клиента

---
## ⚠️ Внимание!

Если приложение не находит ни одного рабочего прокси — скорее всего ваш провайдер режет или фильтрует трафик через DPI (Deep Packet Inspection).
Из-за этого парсер не может нормально достучаться до серверов для проверки доступности и таймаута прокси.

### Временное решение

Чтобы парсинг работал корректно:

1. Включите VPN
2. Запустите парсинг в приложении
3. Дождитесь окончания проверки прокси
4. При необходимости VPN можно отключить — найденные MTProto-прокси обычно продолжают работать нормально

На данный момент это самый стабильный и надёжный способ, пока я экспериментирую с альтернативными методами проверки и верификации прокси, менее зависимыми от DPI-фильтрации.

## 🐧 Запуск на Linux

Во избежание ошибок конфликта пакетов (PEP 668), скрипт необходимо запускать через виртуальное окружение.

### 1. Подготовка системы
Убедитесь, что установлен Python 3 и библиотека Tkinter:
* **Debian / Ubuntu:** `
```bash
sudo apt install python3 python3-venv python3-tk`
```
* **Arch / CachyOS:** `
```bash
sudo pacman -S python python-pip tk`
```
### 2. Установка и первый запуск
Откройте терминал в папке со скриптом и выполните:

```bash
# Создаем изолированную среду
python3 -m venv env
```

### Активируем её (если используете Fish shell, пишите: source env/bin/activate.fish)
* **Debian / Ubuntu:** `
```bash
source env/bin/activate
```
* **Arch / CachyOS:** `
```bash
source env/bin/activate.fish
``` 
### Ставим нужные библиотеки
```bash
pip install customtkinter requests telethon
```
# Запускаем сканер
```bash
python MTProto_Harvester.py
```
🚀 Последующие запуски
Вам больше не нужно ничего скачивать. Просто активируйте среду и запускайте:

```bash
source env/bin/activate
python MTProto_Harvester.py
```
